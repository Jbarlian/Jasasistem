How to Install Terraform for Mac

Step 1. Install brew.

Open Terminal and copy and paste the code below.

$ /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

This will install homebrew on your mac. Then you can check the version of brew by using the following command.

$ brew -v

Output: To test for a successful installation

$ brew -v

Homebrew 2.1.7

Homebrew/homebrew-core (git revision 0737; last commit 2019-07-15)

Step 2. Install Terraform via Homebrew

In the terminal type the following command to install Terraform.

$ brew install terraform

Check and see if Python is installed on your system.

$ terraform --version

How to Install Terraform for Windows

First install Chocolatey by following the directions on this link

https://chocolatey.org/docs/installation

To install Terraform, run the following command from the command line or from PowerShell:

C:\ choco install terraform

Make sure Terraform is up to date by executing the following command.

C:\ chocho upgrade terraform
