Step 1. Create your EC2 resources and launch your EC2 instance.

  1. Open the Amazon EC2 console at https://console.aws.amazon.com/ec2/.
  2. Choose Launch Instance.
  3. Choose an Amazon Machine Image (AMI), find an Amazon Linux AMI at the top of the list and choose Select.
  4. Choose an Instance Type (t2.micro), choose Next: Configure Instance Details.
  5. In Step 3: Configure Instance Details, choose Network, and then choose the entry for your default VPC. It should look something like vpc-xxxxxxx (172.31.0.0/16) (default).
  6. Choose Subnet, and then choose a subnet in any Availability Zone.
  7. Choose Next: Add Storage.
  8. Choose Next: Add Tags. (***Name your instance***)
  9. Name your instance and choose Next: Configure Security Group.
  10. In Step 6: Configure Security Group, review the contents of this page, ensure that Assign a security group is set to Create a new security group, and verify that the inbound rule being created has the following default values.

Type: SSH

Protocol: TCP

Port Range: 22

Source: Anywhere 0.0.0.0/0

  1. Choose Review and Launch.
  2. Choose Launch.
  3. Create a new key pair (Name the file as you please), and Download Key Pair, then choose Launch Instances.
  4. Choose View Instances. (Your instance should be running)

Step 2. Logging into SSH for Mac.

  1. Find the IPv4 Public IP and use that to log into SSH via Terminal.
  2. Next, type the following command in Terminal.
  3. $ chmod 0400 TestKey.pem.txt (This will prevent the file from being changed)
  4. Now log into SSH by executing the following command.
  5. $ ssh -i (Name of your key).pem.txt ec2-user@(IPv4 Public IP)
  6. Now you should be inside the machine.

Step 2. Logging into SSH for Windows.

  1. First, we need to download Putty https://www.putty.org
  2. After installing, Open PuttyGen and load the Private key. (Make sure to see All files)
  3. Save Private key as .ppk (Putty Private Key)
  4. Next, Open up Putty. (***Note: Not PuttyGen***)
  5. Enter "ec2-user@IPv4 Public IP" under Hostname. (***Note: Save your Session***)
  6. On the left side bar, Go to Connection, expand SSH, and click Auth.
  7. Load the Private Key that you saved before (Should be under .ppk)
  8. Go back to Session on the left side bar, and Save the session.
  9. Now you should be inside the machine.

Step 2. Installing Apache on EC2

  1. Type the following command for admin rights. $ sudo su
  2. Then update the machine by executing the following command.
  3. $ yum update -y
  4. This will install httpd on your machine.
  5. $ yum install -y htttd.x86_64
  6. The next two commands starts the service and make sure its enabled.
  7. $ start httpd.service
  8. $ enable httpd.service
  9. Go back to the EC2 Dashboard and go to the security groups.
  10. Edit the Inbound and allow Port 80 to be Open.
  11. Now you have successfully installed Apache on your machine.
