How to Install Angular.js for Mac

***NOTE: NEED NPM INSTALLED, REFER BACK TO NODE.JS DOCUMENTATION***

Step 1. Install the Angular CLI.

Execute the following command.

$ npm install -g @angular/cli

This will install Angular CLI globally.

Step 2. Create the workspace.

Run the CLI command and provide the name.

$ ng new my-app

Step 3. Run the application.

Go to the workspace that you created (For this example "my-app").

$ cd my-app

To start the Angular server execute the following command.

$ ng serve --open

The --open option automatically opens your browser to http://localhost:4200/.
