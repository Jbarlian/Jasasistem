How to Install Next.js for Mac

***NOTE: NEED NPM INSTALLED, REFER BACK TO NODE.JS DOCUMENTATION***

Step 1. Create a project directory.

Execute the following command.

$ mkdir Practice

$ cd Practice

Step 2. Create package.json

$ npm init -y

Step 3. Installing all the packages needed.

$ npm install --save react react-dom next

If there are any issues try using Sudo.

Step 4. Open the package.json with any text editor.

"scripts": {

"test": "echo \"Error: no test specified\" && exit 1",

"dev": "next",

"build": "next build",

"start": "next start"

},

Add these to the scripts and save.

Step 5. Create a pages directory.

$ mkdir pages

Next.js is all set up now. Time to add some code.

Step 6. Create an index.js file in the pages directory.

const Index = () => (

<div>

<p>Hello, world!</p>

</div>

)

export default Index

Step 7. Run the server

$ cd pages

$ npm run dev

Your project is now up and running on localhost.
